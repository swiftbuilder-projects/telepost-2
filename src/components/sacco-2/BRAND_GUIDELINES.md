# Telepost Sacco Brand Guidelines

## Color Palette

The detailed color codes for the Telepost Sacco website are as follows:

- **Telepost Green**: `#008000`
  - Usage: Primary buttons, icons, accents, text highlights.
- **Telepost Yellow**: `#fff503`
  - Usage: Backgrounds (e.g., behind product cards), badges, highlights.
- **Telepost Red**: `#ff0000`
  - Usage: Urgent CTAs, error states, specific section backgrounds (if requested), underlines.

## Typography

- **Font Family**: Nunito Sans
- **Weights**:
  - Bold: 700
  - Extra Bold: 800
  - Black: 900
- **Sizes**:
  - Section Titles: 30px (text-3xl)
  - Hero Titles: 5xl - 8xl

## Notes

- Ensure that these exact hex codes are used. Do not use generic 'red', 'green', or 'yellow' CSS names.
- Do not use variations like `#2D8E4E` (Old Green) or `#E63946` (Old Red).
