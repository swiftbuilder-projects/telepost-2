# Components Directory

This directory will contain all reusable React components for the Telepost Sacco template.

## Structure

Components are organized by category for easier navigation:

```
components/
├── layout/           # Navigation, header, footer
│   ├── Navbar.tsx
│   └── Footer.tsx
├── sections/         # Major page sections
│   ├── Hero.tsx
│   ├── Products.tsx
│   ├── Features.tsx
│   └── ...
├── blocks/           # Reusable content blocks
│   ├── Card.tsx
│   ├── Accordion.tsx
│   └── ...
├── forms/            # Form components
│   ├── Input.tsx
│   ├── Select.tsx
│   └── ...
└── ui/               # Basic UI elements
    ├── Button.tsx
    ├── Modal.tsx
    └── ...
```
